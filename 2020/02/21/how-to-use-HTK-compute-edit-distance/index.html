HTK的下载地址为： http://htk.eng.cam.ac.uk/download.shtml

请注意，下载的时候需要先注册账号然后进行下载。在安装之前，请务必确定已经安装了Visual Stuio，我使用的是Visual Studio 2019，所以推荐使用2019及以上版本。

二、使用HResults模块计算WER

WER的计算公式为：

WER = (S + D + I ) / N = (S + D + I ) / (S + D + C)

其中，S为替换的字数，D为删除的字数，I为插入的字数，C为正确的字数。

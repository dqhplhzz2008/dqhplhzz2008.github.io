rotate image

题目描述

You are given an n x n 2D matrix representing an image.Rotate the image by 90 degrees (clockwise).Follow up:
Could you do this in-place?

题目解析

旋转90°，可以当做两步：第一步是关于对角线取对称；第二步是关于水平线取上下对称。
记住这个原理。

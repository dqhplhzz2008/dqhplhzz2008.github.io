Leetcode 322：零钱兑换

题目描述

给定不同面额的硬币 coins 和一个总金额 amount。编写一个函数来计算可以凑成总金额所需的最少的硬币个数。如果没有任何一种硬币组合能组成总金额，返回 -1。

示例

示例1：

输入: coins = [1, 2, 5], amount = 11输出: 3
解释: 11 = 5 + 5 + 1

示例2：

输入: coins = [2], amount = 3输出: -1

解题思路

这个问题很明显是一种0-1背包问题，利用动态规划的思想，写出其状态转移方程为：

dp(n)=0,if n=0;
dp(n)=min(dp(n),dp(i-coin)+1),n>0
